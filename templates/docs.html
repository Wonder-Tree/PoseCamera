PoseCamera

Realtime Human Pose Estimation

Use following curl command to send image using http request 

      curl -v -F 'upload=@\"./temp/file.jpg\"' http:/localhost:8080

Response (JSON type) will contain list of keypoints of each human body in image. 

      [
        [
          [
            367,
            114
          ],
          [
            359,
            222
          ],
          ...
        ],
        [
         ...
        ]
      ]

Each array contains [x, y] coordinated of each keypoint and keypoints are ordered according to following list

[nose, neck,right shoulder, right elbow, r wrist, left shoder, left elbow, left wrist,right hip, right knee, right ankle, left hip, left knee, left ankle,right eye, left eye,right ear, left ear]
